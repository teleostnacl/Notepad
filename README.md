## 最近更新
### [【美】戴尔·卡耐基《人性的弱点》](图书笔记/【美】戴尔·卡耐基《人性的弱点》.md)
- 你不论断他人，他人就不会论断你。
- 尖刻的批评和斥责几乎总是无济于事的。
- 100次中有99次，没有人会责怪自己任何事，不论他错得多么离谱。
- 做错事的人只会责怪别人 而从来不会责怪自己。
- 当我们指责和纠正某个人的错误的时候，他可能会为自己辩护，反咬我们一口。
- 我们不要责怪别人，我们要试着了解他们，我们要试着明白他们为什么会那样做，尽量设身处地地为他们着想。这比批评更有益处，也更有意义得多；而这也孕育了同情、容忍以及仁慈。
- 我们极希望获得别人的赞扬，同样地，我们也极为害怕别人的指责。
- 批评，并不能够改变事实，反而常常会引起愤怒。
- 责怪别人是愚蠢的行为。我不责怪上帝对智慧分配不均，因为要克服我自己的缺陷，都已经非常困难了。

### [杂志摘录](杂志摘录.md)
- 世界变好时，我们看待世界的眼光会更加苛刻，这可能导致我们得出错误的结论，认为世界其实并没有变好。
- 人是生活的囚徒，要么因为依靠他人而受到囚禁，要么因为孤独而受到囚禁。
- 人生往往是开始各种腾挪变化，到了最后才发现，如果能一成不变，一成不变其实就是最好的结局。
- 成熟的爱是在保持自己的尊严和个性前提下的结合。
- 我理想的婚姻里，有独立的你，有独立的我，也有更好的我们。
- 禀赋效应就是一个典型例子。一旦我们拥有了某件物品，心理上就会觉得它的价值比没拥有时更高，所以，拥有前愿意为它支付的价格和拥有后愿意出售的价格就会不一致。这背后就是人们普遍存在的“损失厌恶”心理，获得与损失在心理上是不对称的，同等损失带来的痛苦要高于同等获得带来的快乐。
