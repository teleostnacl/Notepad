- 我既没有愁苦到成为诗人，也没有冷漠到像个哲学家，但我清醒到足以成为一个废物。

- 人的内心是最大的世界，背离了真正的感情，所有的东西都会变得不真实。
- 做人要学习留白，就能游刃有余，如鱼得水；做事要学习飞白，就能水到渠成，手到擒来。
- 在一个不真实的世界里，人生变成了一场空幻的游戏，人永远只能向前跑，不能向后看，因为看到的都是陌生之事和废墟。
- 生命是这样亲历亲为的事情，每一次轻轻地擦身而过，即使只是一瞬间，都微微地改变了我们。
- 脆弱的人会四处诉说自己的不幸，坚强的人会不动声色地变强大。
