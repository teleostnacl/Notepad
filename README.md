# 手帐札记本
> 这只是一份简单的笔记本吖！

阅读，最高的境界是“消化”，不是去消化自己所读到的内容，而是通过阅读，去消化那些已经发生在自己身上的人生内容。

这是在阅读过程中摘抄了一些有趣的语句的收集本，也是可以时常拿出来去重新品读名言警句的魅力的笔记本，不断的汲取书籍中的养分，时常给自己充充电。
