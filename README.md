- 在自己喜欢的女人面前，每个男人都会变成呆子的。

- 无论多么深的悲哀和痛苦，日久也会淡忘的，“忘记”，本就是人类所以能生存的本能之一。
- 一个人在饥饿、疲倦时，肉体不支，精神更脆弱，内贼已将生，外贼自然更容易乘虚而入。
- 水，看来虽柔和平静，其实却是无坚不摧，无物可挡的，滴水已能穿石，洪水更能使山岳移形，城市毁灭，自古以来，天下就从来没有任何一种东西能抵抗水的力量。
